1) Download the project folder in your home directory.
2) Move the project folder to your /usr/bin directory by executing this command:
	sudo mv naila /usr/bin
3) Go to this location by using this command:
	cd /usr/bin/naila
4) Make shell script executable by this command:
	chmod +x script.sh
5) Execute the shell script by this command:
	./script.sh
6) Now you can use naila hasslefree.
	example: naila create a folder hello
		 naila create a file aditi
		 naila what is the time
		 and many more...
